# 文件整理技能 (File Organizer Skill)

一个用于 Claude Code 的文件整理技能，帮助用户自动分类、重命名和整理文件。

## 简介

这个技能可以智能地整理您的文件目录，支持多种分类方式和重命名规则，让文件管理变得轻松高效。

## 功能特性

### 🗂️ 多种分类方式
- **按扩展名分类** - 自动识别文件类型并归类（图片、文档、视频、音频、压缩文件、代码等）
- **按文件大小分类** - 根据文件大小自动分组（极小、小、中、大、超大）
- **按修改日期分类** - 按年/月/日/季度组织文件
- **按文件名前缀分类** - 提取文件名前缀自动分组

### ✨ 灵活的重命名选项
- 保持原文件名
- 追加日期时间戳
- 追加序号（全局或分类独立）
- 组合时间戳和序号

### 🛡️ 安全可靠
- 默认使用复制模式，保留原文件
- 多种文件冲突处理策略
- 移动前确认提示

## 使用方法

### 基本用法

```
整理 Downloads 文件夹
```

```
帮我整理这个文件夹，按文件类型分类
```

```

```

## 支持的文件类型

### 图片 (9种格式)
jpg, jpeg, png, gif, bmp, svg, webp, ico, tiff

### 文档 (11种格式)
pdf, doc, docx, txt, rtf, odt, xls, xlsx, ppt, pptx, csv

### 视频 (8种格式)
mp4, avi, mkv, mov, wmv, flv, webm, m4v

### 音频 (7种格式)
mp3, wav, flac, aac, ogg, m4a, wma

### 压缩文件 (6种格式)
zip, rar, 7z, tar, gz, bz2

### 代码文件 (16种格式)
py, js, ts, html, css, java, cpp, c, h, php, rb, go, rs, json, xml, yaml, sql

## 文件大小分类

| 分类 | 大小范围 |
|------|----------|
| 极小 | < 100KB |
| 小 | 100KB - 1MB |
| 中 | 1MB - 10MB |
| 大 | 10MB - 100MB |
| 超大 | > 100MB |

## 文件冲突处理

当目标文件已存在时，可选择：

- **跳过（默认）** - 保留目标文件，跳过当前文件
- **覆盖** - 用源文件替换目标文件
- **自动重命名** - 自动添加后缀 `_1`, `_2`, `_3`...
- **询问用户** - 每次冲突时询问如何处理

## 配置选项

### 可选参数
- **源文件夹路径** - 需要整理的目录
- **目标文件夹路径** - 整理后存放的目录（默认：organized_files）
- **分类方式** - 扩展名/大小/日期/前缀
- **重命名方式** - 时间戳/序号/保持原名
- **操作类型** - 复制/移动
- **冲突处理** - 跳过/覆盖/重命名/询问

## 技术细节

- 支持 Windows、Linux、macOS
- 使用 bash 命令进行文件操作
- 支持嵌套目录结构
- 最大重命名尝试次数：1000

## 注意事项

1. **移动模式**：使用移动模式会删除原文件，请谨慎操作
2. **覆盖模式**：选择覆盖会导致同名文件被永久替换
3. **权限问题**：确保对源目录和目标目录有读写权限
4. **特殊字符**：文件名包含特殊字符时建议使用自动重命名
